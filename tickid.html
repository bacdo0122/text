HTML

HTML viết tắt của Hypertext Markup Language là ngôn ngữ lập trình dùng để xây dựng và cấu trúc lại các thành phần có trong Website. HTML tạm dịch là ngôn ngữ đánh dấu siêu văn bản. Người ta thường sử dụng HTML trong việc phân chia các đoạn văn, heading, links, blockquotes,…

Các nội dung liên quan đến HTML

  • - Nhóm thẻ về tiêu đề
  • - Nhóm thẻ về định dạng văn bản
  • - Nhóm thẻ về hiển thị văn bản
  • - Nhóm thẻ về liên kết và hiển thị ảnh
  • - Nhóm thẻ về danh sách
  • - Nhóm thẻ về table
  • - Nhóm thẻ về form

Các kiểu hiển thị của nhóm thẻ HTML

  • - Kiểu inline:Hiển thị hàng
    ví dụ:a, nhóm các thẻ định dạng văn bản,...
  • - Kiểu inline:Hiển thị cột
    ví dụ:div, p, ol, li, ul,...

CSS

CSS là chữ viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra bởi các ngôn ngữ đánh dấu (ví dụ như HTML). CSS sẽ giúp chúng ta có thể thêm một chút “phong cách” vào các phần tử HTML đó như đổi màu sắc trang, đổi màu chữ, thay đổi cấu trúc,…rất nhiều.

Các thuộc tính liên quan đến CSS

  • - Thuộc tính Color
  • - Thuộc tính Background và Border
  • - Thuộc tính Table
  • - Thuộc tính Basic Box
  • - Thuộc tính bố cục Flexible Box
  • - Thuộc tính Text
  • - Thuộc tính Font

Các kiểu hoạt hình của Css

  • - Kiểu Animation: Viết trong @keyframes
    gồm các thuộc tính
    (delay, time-function, name,...)
  • - Kiểu Transition:Thay đổi thuộc tính
    gồm các thuộc tính
    (thuộc tính, delay,time-function,...)

jQuery

jQuery is a lightweight, "write less, do more", JavaScript library The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code. jQuery also simplifies a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.

The jQuery library contains the features

  • - HTML/DOM manipulation
  • - CSS manipulation
  • - HTML event methods
  • - Effects and animations
  • - AJAX
  • - Utilities

The biggest companies use jQuery

  • - Google
  • - Microsoft
  • - IBM
  • - Netflix
